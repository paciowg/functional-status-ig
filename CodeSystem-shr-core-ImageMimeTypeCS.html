Image Mime Type CS

Mime types that apply to photographic images.

This code system http://hl7.org/fhir/us/PACIO-functional-status/CodeSystem/shr-core-ImageMimeTypeCS defines the following codes:

CodeDisplayDefinition

image/gif .gif.gif
image/jpeg .jpeg.jpeg
image/png .png.png
